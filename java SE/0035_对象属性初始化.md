# 属性初始化

## 对象属性初始化

对象属性初始化有三种方法

1. 声明该属性的时候初始化
2. 构造方法中初始化
3. 初始化块

```
public class Hero {
	//声明的时候初始化
	public String name="some hero";
	
	protected float hp;
	
	float maxHP;
	{
		maxHP=200;
	}
	public Hero() {
		hp=100;
	}
}
```
## 类的属性初始化

类属性初始化有2种

1. 声明该属性的初始化
2. 静态初始化块

```
public class Hero {
	public String name;
	protected float hp;
	float maxHP;
	
	//物品栏的容量
	
	static {
		itemCapacity=9;//静态初始化块
	}
	public static int itemCapacity=6;
	
	public Hero() {
		
	}
	
	public static void main(String[] args) {
		System.out.println(Hero.itemCapacity);
	}
}
```
