# 换源
```
sudo su     #输入密码
cd /etc/apt  #切换到apt源文件
mv  sources.list sources.list_bak  #备份源文件
nano sources.list  #新建一个，然后将下面的内容copy进去
```

更新配置
按ecs，然后:wq保存后退出
再执行命令更新配置
```
sudo apt-get update
```
